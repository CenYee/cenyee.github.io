sublime教程

安装sublime

安装时勾选Add to explorer context menu

添加Sublime Text到环境变量：

Win + R 运行 sysdm.cpl 打开 “系统属性”，在系统变量中增加Sublime Text的安装目录，如 D:\Program Files\Sublime Text 3

后面就可以利用subl命令使用Sublime Text：

安装Package Control

Sublime Text 支持大量插件，使用Package Control可以方便的浏览、安装和卸载Sublime Text中的插件。

按键ctrl+shift+p打开sublime的控制台，输入Install Package Control，回车安装。

查看已经安装的插件：
ctrl+shift+p打开控制台，输入list packages.

查看可更新的插件：
ctrl+shift+p打开控制台，输入upgrade packages.

查看可卸载的插件：
ctrl+shift+p打开控制台，输入remove package.

其他

安装插件流程：
Ctrl+Shift+P打开控制台，Package Control: Install Package打开搜索框。
